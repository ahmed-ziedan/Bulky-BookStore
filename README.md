# 📚 Bulky BookStore

A full-featured e-commerce web application built with ASP.NET Core MVC (.NET 8).

## 🚀 Features

- Browse & purchase books online
- Shopping cart & order management
- Stripe payment integration
- Role-based authorization (Admin / Employee / Company / Customer)
- Admin dashboard to manage products, categories & orders
- Facebook & Microsoft social login
- Email notifications via SendGrid

## 🛠️ Tech Stack

- ASP.NET Core MVC (.NET 8)
- Entity Framework Core (Code First)
- ASP.NET Core Identity
- Repository Pattern & Unit of Work
- Stripe API
- SendGrid
- TinyMCE
- Bootstrap 5

## ⚙️ Getting Started

1. Clone the repository
2. Update `appsettings.json`
3. Fill in your credentials
4. Run `Update-Database` in Package Manager Console
5. Run the project
